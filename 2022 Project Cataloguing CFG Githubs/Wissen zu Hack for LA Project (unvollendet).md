Henriette hat 2022 ein Email bekommen von Bonnie Wolfe von Code for America / Hack for LA. Auszüge:

"I am the Executive Director of Hack for LA, which is Code for America's Los Angeles Chapter. We have a project that is seeking to map all the Civic  Hacking Organizations in the world with GitHub code repositories and connect them so that people working on solutions to similar problems can find each other's GitHub repositories.  It's called [__civictechindex.org__](http://civictechindex.org)  
We are finding it a bit challenging to figure out where Code for Germany and Open Knowledge Foundation Germany chapters fit in the global ecosystem. I hope you can help."

=> sie wollten wissen, ob sie Code for Germany unter Code for All oder unter OKF De reihen sollten.  
=> Sonja hat am Mittwoch, 27. April 2022 kurz mit Bonnie gezoomt und dadurch mehr übers Projekt erfahren.  
=> Bonnie ist auch im Code for All Slack mit @‌bonnie

Das Projekt soll Ende Mai 2022 fertig werden, und kann dann für unsere Katalogisierung hilfreich sein!

Update: mit April 2023 ist das Projekt noch nicht fertig. Auf der Webseite von Hack for LA ([Projects by Hack for LA](https://www.hackforla.org/projects/) ) ist es als “on hold” dargestellt. Auf der Projektwebseite ([Civic Tech Index](http://civictechindex.org/) ) steht “coming soon.”
